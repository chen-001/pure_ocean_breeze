## 🌟初始化
* 在初次安装框架时，请进行初始化，以将路径设置到自己的文件里
* 使用如下语句进行初始化
```python
import pure_ocean_breeze.initialize.initialize
pure_ocean_breeze.initialize.initialize.initialize()
```
* 然后根据提示进行操作即可（⚠️ 请注意路径不要写反斜杠`\`，而要写成`/`）
* 经过初始化后，以后就可以直接使用，不论重启电脑或者版本升级，都不用再初始化

⚠️  如果更换了数据库路径，请重新初始化
